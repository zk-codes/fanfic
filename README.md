# Fanfic Writers Collective

A web directory of folks with personal sites who write and publish fanfiction.

## Submission Guidelines

To be included in the Fanfic Writers Collective:

- You must write and publish fanfiction
- It must be your personal site or blog (your fanfic can be on another domain/subdomain)
- The site should reflect something about yourself

## How To Add Your Site

### Option 1: Submit Via Pull Request

1. Fork this repository
2. Edit `index.html` to add your entry in alphabetical order by name
3. Follow the HTML format below
4. Submit a pull request

**HTML Format:**
```html
<li>
    <a href="DOMAIN" rel="noopener">NAME</a> 
     | FANDOMS
</li>
```

### Option 2: Email

Email me via hi@zacharykai.net with your name, site URL, and fandom/s.
